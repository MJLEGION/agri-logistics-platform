## 🎬 Animation Flow

### When Landing Page Loads:

```
1. Page loads → Background visible
2. Hero section fades in
3. Content sections appear from bottom
4. Cards have subtle entrance animation
5. Section titles fade in with slide-up effect
```

### When User Scrolls:

```
1. Testimonials scroll up
2. How It Works section appears
3. Each step card slightly scales/fades in
4. Farmer cards appear with stagger effect
5. View All button appears
```

## 📊 Data Display Examples

### Step Card Data Flow:

```
Data (recommendedFarmers.ts):
{
  step: 1,
  icon: 'people',
  title: 'Create Your Account',
  description: 'Sign up...',
  color: '#2ECC71'
}
        ↓
Component (HowItWorksCard.tsx):
- Creates step circle with icon
- Displays step number and color
- Shows title and description
        ↓
UI Output:
[🧑‍🤝‍🧑] Step 1
Create Your Account
Sign up in just 2 minutes...
```

### Farmer Card Data Flow:

```
Data (recommendedFarmers.ts):
{
```

## 🎯 User Journey

### First-Time Visitor:

```
1. Lands on page → Sees hero section
2. Scrolls down → Reads "How We Serve You"
3. Continues scrolling → Sees "Who We Serve"
4. More scrolling → Reads testimonials
5. Key moment → Sees "How It Works" (NEW)
   → Understands the exact process
   → Feels confident to sign up
6. More scrolling → Meets recommended farmers (NEW)
   → Sees real, verified people
   → Builds trust in platform
7. Bottom → Calls to action
```
