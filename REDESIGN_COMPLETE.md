## 🔧 Customization Guide

### **Change Colors**

Edit `src/config/theme.ts`:

```typescript

  primary: "#YOUR_COLOR", // Main brand color
  background: "#FFFFFF", // Keep white for Triad style
  text: "#2C3E50", // Professional text
  // ... other colors
};
```
