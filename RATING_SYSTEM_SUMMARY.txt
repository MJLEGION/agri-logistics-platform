🎯 REQUIRED PARAMETERS TO NAVIGATE:

To launch the rating screen, you MUST provide these parameters:

  transactionId    ← Unique ID for this trip/transaction
  transporterId    ← ID of the person being rated
  transporterName  ← Display name of the person being rated
  farmerId         ← Your (shipper's) user ID
  farmerName       ← Your (shipper's) display name

EXAMPLE:

  navigation.navigate('Rating', {
    transactionId: 'trip-12345',
    transporterId: 'trans-789',
    transporterName: 'John Transport Co.',
    farmerId: 'farmer-456',
    farmerName: 'Ahmed Hassan',
  });
